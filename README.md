# SecurityOauthKnpu Module
[![Latest Stable Version](https://poser.pugx.org/spryker/security-oauth-knpu/v/stable.svg)](https://packagist.org/packages/spryker/security-oauth-knpu)
[![Minimum PHP Version](https://img.shields.io/badge/php-%3E%3D%208.3-8892BF.svg)](https://php.net/)

SecurityOauthKnpu integrates the https://github.com/knpuniversity/oauth2-client-bundle with Spryker's security layer. It provides the glue between KnpU's OAuth2 client abstraction and Spryker's identity resolution and persistence.

## Installation

```
composer require spryker/security-oauth-knpu
```
